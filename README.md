# 🎵 Spotify Downloader GUI

A lightweight, modern Python desktop application built with CustomTkinter that allows you to easily download music from Spotify. 

### Key Features:
* **All-in-One Download:** Paste links to tracks, albums, or entire playlists.
* **Full Metadata Integration:** Downloads include high-quality cover art, artist names, album info, and correct tags.
* **Smart Bypass:** Integrated cookie support (`cookies.txt`) to easily bypass YouTube's "Sign in to confirm you're not a bot" detection.
* **Asynchronous Logs:** Monitor progress in real-time through the built-in terminal logs without freezing the app window.
* **One-Click FFmpeg Setup:** Auto-installs FFmpeg directly inside the application folder.
